# vpn_easy — Windows VPN engine artifacts

This directory contains the headers and pre-built binaries for `vpn_easy.dll`,
the Windows VPN engine compiled from the [TrustTunnelClient](https://github.com/TrustTunnel/TrustTunnelClient) source.

## Directory layout

```
vpn_easy/
  include/
    vpn_easy.h          C API header (copied from TrustTunnelClient)
    vpn/
      platform.h        WIN_EXPORT macro definition
  lib/
    vpn_easy.lib        Import library (generated by MSVC build) — NOT committed
  bin/
    vpn_easy.dll        Windows DLL — NOT committed
```

## How to build vpn_easy.dll

On a Windows x64 machine with Visual Studio 2022 and Conan 2 installed:

```bat
cd TrustTunnelClient
conan install . --build=missing -s os=Windows -s arch=x86_64 -s build_type=Release
cmake --preset conan-release
cmake --build --preset conan-release --target vpn_easy
```

Then copy the outputs here:
```
build/Release/vpn_easy.dll  →  plugins/vpn_plugin/windows/vpn_easy/bin/vpn_easy.dll
build/Release/vpn_easy.lib  →  plugins/vpn_plugin/windows/vpn_easy/lib/vpn_easy.lib
```

## Runtime requirement

`vpn_easy.dll` requires **Wintun** to be installed on the target machine.
Download from https://www.wintun.net/ and place `wintun.dll` alongside `vpn_easy.dll`
in the application directory, or install it system-wide.

The Flutter app must be run as Administrator (or have the `SeLoadDriverPrivilege`)
for Wintun to create the TUN adapter.
